# 🔐 AI-Based Password Strength Analyzer & Security Visualizer

An intelligent password strength analyzer that evaluates password security using entropy calculation, pattern detection, and crack-time estimation. This project provides real-time feedback and visual indicators to help users create strong and secure passwords.

---

## 🚀 Features

- 🔍 Real-time password strength analysis  
- 🧠 Entropy-based security calculation  
- ⚠️ Detection of weak patterns (e.g., "1234", "password")  
- ⏱️ Crack-time estimation  
- 📊 Visual strength meter (Weak → Strong)  
- 🎨 Clean and responsive UI
